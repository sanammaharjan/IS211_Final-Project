About Blogging App

This is IS 211 - Final Project and blogging app developed using python, HTML, CSS, Bootstrap & Javascript. Blog layout is sourced from Styleout and modified by Sanam Maharjan.

In this app, only registerd user are allowed to post and manage the blog. Please check login page to register and login to admin dashboard. Users are allowed to view the post and comment it once they click on the post and view it individually.
